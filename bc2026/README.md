# Cero Presupuesto, Producto Top

Slides for the talk at **BC Day ES 2026** · Microsoft Madrid · 14 April 2026

Built with [Slidev](https://sli.dev/).

## Setup

```bash
npm install
```

## Development

Start the dev server with live reload:

```bash
npm run dev
```

The presentation will be available at http://localhost:3030

## Build

Build for production (static SPA):

```bash
npm run build
```

Output goes to `dist/`.

## Export

Export slides to PDF:

```bash
npm run export
```
